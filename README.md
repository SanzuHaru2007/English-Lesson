Простой Python-проект для тренировки английского языка с помощью голосового распознавания.

## 📌 Описание
Это консольная игра, в которой пользователь тренирует произношение английских слов или перевод слов с русского на английский.  
Программа записывает голос пользователя, распознаёт сказанное слово и проверяет правильность ответа.
Проект использует словарь слов по уровням сложности: easy, medium и hard.  
За правильный ответ начисляется 1 очко, а после 3 ошибок игра заканчивается.

## 🎮 Режимы игры
В игре есть два режима:

1. **Произношение английских слов**  
   Пользователь видит английское слово и должен правильно его произнести.

2. **Перевод с русского на английский**  
   Пользователь видит русское слово и должен сказать его перевод на английском.

## 🧠 Уровни сложности
- `easy` — простые слова
- `medium` — слова средней сложности
- `hard` — сложные слова

## 🛠️ Используемые библиотеки
Для работы проекта используются:

- `sounddevice`
- `numpy`
- `scipy`
- `speech_recognition`
- `random`

## 🚀 Установка и запуск
1. Склонируйте репозиторий:
Перейдите в папку проекта:
cd project-name
Установите зависимости:
pip install sounddevice numpy scipy SpeechRecognition
Запустите игру:
python game.py

📋 Как играть
Выберите уровень сложности:
easy, medium или hard
Выберите режим игры:
1 - Произношение английских слов
2 - Перевод с русского на английский
После команды программы произнесите слово в микрофон.
Программа распознает речь и проверит ответ.

🏆 Система очков
За правильный ответ игрок получает +1 очко.
За неправильный ответ начисляется ошибка.
После 3 ошибок игра заканчивается.
